# Dashboard

The first thing you see after logging in. Shows a high-level overview of what's going on across all your servers.

At the top there are five stat cards — total players online, CPU usage, how many servers are running (out of your total profiles), active mod count, and when the last player joined. Pretty self-explanatory stuff.

## Live leaderboard

If anyone's playing, a leaderboard pops up with all connected players sorted by score. Top 3 get medals. Each player's SteamID links to their Steam profile. Bots show up too but they're tagged.

## Server cards

Each running profile gets a card showing the map, player count, uptime, and port numbers. If nothing's running you just see an empty state telling you to go start something from Server Control.

All of this refreshes automatically — you don't need to manually reload.
